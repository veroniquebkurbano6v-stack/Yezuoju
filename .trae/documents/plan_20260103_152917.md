## 创建 .gitignore 文件

### 需要排除的文件和目录

#### Python 相关
- `venv/` - Python 虚拟环境
- `__pycache__/` - Python 字节码缓存
- `.mypy_cache/` - mypy 类型检查缓存
- `*.pyc` - 编译的 Python 文件
- `*.pyo` - 优化的 Python 字节码
- `*.pyd` - Python 动态库

#### 前端相关
- `frontend/node_modules/` - npm 依赖
- `frontend/dist/` - 构建输出
- `frontend/.DS_Store`

#### 系统和 IDE
- `.DS_Store` - macOS 系统文件
- `Thumbs.db` - Windows 缩略图缓存
- `.idea/` - PyCharm IDE 设置
- `.vscode/` - VS Code 设置（保留 extensions.json）
- `*.swp` - Vim 临时文件
- `*.swo`
- `*~` - 备份文件

#### 环境配置
- `.env` - 环境变量文件（可能包含 API 密钥）
- `.env.local`

#### 其他
- `dialog_checkpoints/` - 对话检查点文件
- `*.log` - 日志文件

### 保留的文件
- `src/data/` - 项目数据文件（PDF、JSON 等）
- `frontend/src/` - 前端源代码
- `backend/` - 后端源代码
