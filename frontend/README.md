# StoryRag v2.0 Frontend

React + Vite + Tailwind CSS frontend for StoryRag v2.0 intelligent PDF retrieval system.

## Development

```bash
npm install
npm run dev
```

## Features

- Query Panel: Direct vector search with citation display
- Dialog Management: Create, delete, and chat in isolated conversations
- Real-time chat with DeepSeek agent integration
- Responsive design with Tailwind CSS
